# Потрачено (potracheno)

**Telegram-бот, который напоминает о повторяющихся платежах — чтобы не ловить штрафы
и перестать платить за то, чем уже не пользуешься.**

Подписки, аренда, связь, терапевт, тренер, страховки, налоги, садик, курсы — всё это
повторяется с разной периодичностью и в разных валютах. Часть платежей легко забыть
(привет, пени и просрочки), часть давно не нужна, но деньги продолжают списываться.
«Потрачено» собирает все регулярные платежи в одном месте и заранее напоминает о каждом.

---

## Что умеет

- 📝 **Добавление платежей через диалог «вопрос-ответ»** — название, категория, валюта,
  сумма, периодичность, напоминания. Без форм и таблиц.
- 🔁 **Гибкая периодичность** — каждую неделю / несколько раз в неделю / в определённый день
  недели / ежемесячно / несколько чисел в месяц / раз в квартал / раз в год / несколько раз
  в год / разово.
- 💱 **Любые валюты** — $, €, ₽, MXN, ₸ (теньге), ₾ (лари), ֏ (драм), zł (злотый) и любой
  ISO-код. **Основной валюты нет** — у каждого платежа своя; в итогах суммы показываются
  раздельно по каждой валюте, без конвертации.
- ⏰ **До 3 напоминаний на платёж** — за сколько дней предупредить, настраивается. Время
  напоминаний — в твоём часовом поясе.
- ✅ **Действия прямо в напоминании** — «Оплатил», «Отложить», «Изменить», «Больше не плачу».
  Если не ответить — бот считает, что платёж в силе, и продолжит напоминать в следующем цикле.
- 📊 **Статус по запросу** — суммы предстоящих платежей на этой неделе / в этом месяце /
  за 12 месяцев и ближайший платёж (название, сумма, дата, сколько дней осталось).
- ✏️ **Правки платежа** — название, сумма, валюта, периодичность, напоминания; в т.ч. прямо
  из напоминания.
- ✉️ **Обратная связь** — сообщить об ошибке или предложить идею.

---

## Как это работает (коротко)

1. Пользователь проходит короткий онбординг: часовой пояс и удобное время напоминаний.
2. Добавляет платежи мастером «вопрос-ответ». Бот вычисляет дату ближайшего платежа и
   заранее ставит до трёх напоминаний.
3. Раз в минуту планировщик проверяет, какие напоминания пора отправить, и шлёт их в чат с
   кнопками действий.
4. Когда дата платежа проходит (оплачено или просто наступила), бот переносит платёж на
   следующую дату цикла и снова ставит напоминания.

Подробности архитектуры — в [CLAUDE.md](CLAUDE.md).

## Технологии

- **Python 3.12**, [aiogram 3](https://docs.aiogram.dev/) (async, long-polling)
- **SQLite** через SQLAlchemy 2.0 (async, `aiosqlite`)
- **APScheduler** — планировщик напоминаний
- **python-dateutil** — движок периодичности
- Часовые пояса — stdlib `zoneinfo`

Бесплатно и self-hosted: один процесс + файл БД, без платных API.

## Запуск

### 1. Получить токен бота
Напиши [@BotFather](https://t.me/BotFather) → `/newbot` → получи `BOT_TOKEN`.
(Опционально, для приёма обратной связи) узнай свой chat id у
[@userinfobot](https://t.me/userinfobot) — это `ADMIN_CHAT_ID`.

### 2. Настроить окружение
```bash
cp .env.example .env
# открой .env и впиши BOT_TOKEN (и при желании ADMIN_CHAT_ID)
```

### 3a. Запуск через Docker (рекомендуется)
```bash
docker compose up -d
docker compose logs -f
```

### 3b. Запуск локально
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m bot.main
```

База данных создаётся автоматически в `./data/potracheno.db`.

## Деплой на свой сервер (24/7)

Через Docker (`docker compose up -d`) — самый простой путь, том `./data` хранит БД.

Либо как systemd-сервис:
```ini
# /etc/systemd/system/potracheno.service
[Unit]
Description=Potracheno Telegram bot
After=network-online.target

[Service]
WorkingDirectory=/opt/potracheno
ExecStart=/opt/potracheno/.venv/bin/python -m bot.main
EnvironmentFile=/opt/potracheno/.env
Restart=always

[Install]
WantedBy=multi-user.target
```
```bash
sudo systemctl enable --now potracheno
```

Подойдёт и бесплатный Oracle Cloud Free Tier.

## Конфигурация (`.env`)

| Переменная | Назначение | По умолчанию |
|---|---|---|
| `BOT_TOKEN` | Токен от @BotFather (обязательно) | — |
| `ADMIN_CHAT_ID` | Куда пересылать обратную связь | пусто |
| `DB_URL` | Строка подключения к БД | `sqlite+aiosqlite:///data/potracheno.db` |
| `DEFAULT_TZ` | Часовой пояс по умолчанию | `Europe/Moscow` |
| `DEFAULT_NOTIFY_TIME` | Время напоминаний по умолчанию | `10:00` |

## Структура проекта

```
bot/
  main.py            # точка входа: бот, роутеры, планировщик, polling
  config.py          # настройки из .env
  texts.py           # все пользовательские строки (RU)
  callbacks.py       # типизированные callback-data
  keyboards.py       # inline-клавиатуры
  states.py          # FSM-состояния диалогов
  middlewares.py     # сессия БД + автосоздание пользователя
  db/                # модели и доступ к данным (SQLAlchemy)
  handlers/          # онбординг, добавление, список, статус, правки, напоминания, фидбэк
  services/          # money, recurrence, reminders, scheduler, dates, timezones, humanize
tests/               # юнит-тесты ядра
```
